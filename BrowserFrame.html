Class: BrowserFrame

BrowserFrame

new BrowserFrame(config, frame)

Define a basic mozbrowser iframe class. It creates a mozbrowser iframe, and finally returns the DOM element it just created. The BrowserFrame instance has a 'element' field, which is the mozbrowser iframe it generated or converted to.
Parameters:
Name Type Argument Description
config Object The configuration object generated by BrowserConfig.
frame DOMIFRAMEElement <optional>
 The existing frame to be converted to mozbrowsr iframe.
